# 风格化提示词库工作流

## 文件位置

```
prompts/学院纯欲风.md    (~27KB, 15个英文生图模板)
prompts/福利姬-中文模板.md (~22KB, 6个中文模板)
prompts/README.md        (索引与使用流程)
```

## 工作流

1. **确定风格** → 学院纯欲风 or 福利姬
2. **读取文件** → `cat` 或 `read_file` 对应 .md
3. **选择元素** → 服装 + 姿势 + 配饰 + 场景 + 光线各选一项
4. **生图方式**：
   - **A（推荐）**：文件中的英文模板直接 `--prompt`
   - **B**：映射到 `portrait-photography` 模板 vars
   - **C**：翻译为英文用通用 `generate.py`
5. **安全检查** → 禁用词：旗袍/旗装；高风险词按降敏策略处理
6. **直接发原图** → 不裁白边、不生成 JPG 副本

## 模板选择优先级（人物写真类）

1. `portrait-photography`（首选）：单人写真、纯欲、古风、棚拍
2. `bedroom-mirror-selfie`：卧室镜前自拍、居家私房
3. `kpop-idol`：韩团舞台、打歌服
4. `person-photoshoot-3x3`：同人物九宫格
5. `poster-cosplay`：Cosplay 海报（注意 prompt 长度，xxx 要精简）
6. `couple-portrait`：双人写真

## 已验证成功的组合

### 古风汉服 · portrait-photography · 2160×3840
- 服装：薄荷绿+浅黄渐变汉服宽袖外袍、白色刺绣抹胸、金色细链腰饰
- 发型：古典发髻+花朵蝴蝶发饰+竹编斗笠薄纱面纱
- 场景：灰褐色墙面+木质长凳棚拍
- 光线：左前侧方向光
- 姿态：背影回眸 / 端庄坐姿

### 古风短装背影 · portrait-photography · 2160×3840
- 服装：浅粉色短款对襟刺绣短衫（到腰线）、蜜桃橙薄纱短裙
- 姿态：背对镜头回头越过左肩

### 古风性感 · poster-cosplay · 2160×3840
- xxx 精简为：成年东亚女性+酒红丝绒改良旗袍+黑色波浪长发+珍珠耳坠+红唇+折扇遮半脸+深色木质棚拍+暖色侧光
- 注意：xxx 过长会导致所有端点 RemoteDisconnected

## 禁用词

旗袍、旗装 → 改用汉服、对襟短衫、花神裙、齐胸襦裙改良款

## 高分辨率注意事项

- 2160×3840 边缘偶有白线（API 端点行为，忽略即可）
- 88996.cloud 支持原生 2160×3840，bltcy.ai 最高 1024×1536
- 总 prompt 过长时所有端点可能 RemoteDisconnected，需精简
