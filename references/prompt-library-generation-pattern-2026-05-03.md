## 提示词素材库 → 模板映射规则

### 元素提取优先级（从 prompts/ 文件中）

1. **服装**：选一个主类别 + 面料质感 + 颜色
2. **发型**：古风发髻 / 学院马尾 / 私房散发，配合发饰
3. **配饰**：耳饰 + 头饰 + 腰饰（不超过3类，避免信息过载）
4. **道具**：选1个核心道具（团扇/折扇/书本/花束等）
5. **场景**：匹配风格的场景（古风→木质棚拍，学院→教室，私房→卧室）
6. **光线**：选一种方案，用三段式描述（光源→照射方向→效果）
7. **姿态**：从80+姿势中选1个，用中性/艺术化语言描述

### 模板选择

- 单人写真 → `portrait-photography`（首选）
- 卧室自拍 → `bedroom-mirror-selfie`
- 韩团风 → `kpop-idol`
- 九宫格 → `person-photoshoot-3x3`

### 安全语言规范

**安全词（可直接用）：**
- 成年东亚女性、白皙肌肤、精致五官
- 轻薄雪纺、层叠飘逸、垂坠感
- 端庄、舒展、优雅、杂志感、棚拍
- 古典东方气质、空灵梦幻、柔和粉彩

**需替换词：**
- 轻薄透视 → 轻薄雪纺/轻薄面料
- 挑逗/诱惑 → 专注/自信/从容
- 舌尖/触唇 → 手指自然靠近面部
- 私房写真 → 高级人像摄影
- 若隐若现 → 层叠飘逸
