## 快速使用流程

1. **确定风格** → 学院纯欲 or 福利姬
2. **读取文件** → `read_file` 对应 .md 文件
3. **选择元素** → 服装 + 姿势 + 配饰 + 场景 + 光线各选一项
4. **生图方式**：
   - **A（推荐）**：用文件中的英文模板直接 `--prompt`
   - **B**：映射到 `portrait-photography` 模板 vars
   - **C**：翻译为英文用通用 `generate.py`
5. **安全检查** → 按 skill 中的降敏策略处理
6. **生成发送**

## 安全提示词规则

**避免：** 轻薄透视, 纯欲(特定组合), 露点, 挑逗, 舌尖, 私密处具体描写, 直白掀衣/脱衣，旗袍，旗装
**替代：** 光影层次, 面料垂坠感, 若隐若现, 姿态张力, 氛围感
